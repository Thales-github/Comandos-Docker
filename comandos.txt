baixa o container que executa a imagem de teste do docker: docker container run hello-world

baixar uma imagem e rodar container: docker container run debian(tecnologia que eu quero rodar)

baixar um imagem, rodar o container e passar um comando para ser executado dentro do container em seguida: docker container run debian (bash --version) comando que será executado dentro do container

listar container rodando: docker container ps

listar todos os containers independente se está rodando ou não: docker container ps -a

criar container, rodar e apagar após execução: docker container run --rm(flag que indica que deve ser apagado após execução) debian(imagem) bash --version(comando que executa dentro do container)

acessar o terminal do seu container: docker container run -it debian bash(caso você xrie algo no container utilizando o run ele cria novos container e então você perde as informações a cada run executad pois recria o container)

criar container com nome específico: docker container run --name mydeb1 debian
